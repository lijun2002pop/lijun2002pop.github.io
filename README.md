# ifan 的博客

个人技术博客，基于 Hexo 和 insnow 主题构建。

## 📋 项目信息

- **主题**：insnow
- **部署**：GitHub Pages
- **语言**：中文

## 🛠 环境搭建

### 安装依赖
```bash
npm install
```

### 开发命令
- **本地预览**：`hexo server`
- **生成静态文件**：`hexo generate`
- **清理缓存**：`hexo clean`
- **部署到 GitHub**：`hexo deploy`

### 创建新文章
```bash
hexo new "文章标题"
```

## 📁 项目结构
├── source/          # 源文件
    ├── _posts/          # 文章目录
    ├── about/           # 关于页目录
    ├── assets/          # 资产目录（图片、视频等）

├── themes/insnow/       # 主题目录
├── package.json     # 依赖文件
├── README.md        # 项目说明


## 🔧 配置说明

### 全局配置
编辑 `_config.yml` 文件，设置个人信息、主题、部署等配置。

### 主题配置
编辑 `themes/insnow/_config.yml` 文件，设置主题相关配置。

## 📝 写作指南

### 文章格式
```markdown
---
title: 文章标题
date: 2026-03-19 12:00:00
tags: [标签1, 标签2]
categories: 分类
---

文章内容...
```
